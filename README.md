# How-To MT2 General

Collection of tools & tutorials for MT2 development.

This repository contains standalone utilities, scripts, and step-by-step guides accumulated over the years. Each subfolder is a self-contained project with its own documentation.

**[Browse the site](https://martysama0134.github.io/how-to-mt2-general/)**

## Projects

| Project | Category | Description |
|---------|----------|-------------|
| [Client PyLib Check](client-pylib-check/) | Tool | C++20 refactoring for Python library file checking |
| [Eter Group Parser](eter-group-parser/) | Tool | Load, resave, manipulate, and repair Eter Group files |
| [Pack Cleaning](pack-cleaning/) | Tool | Python scripts for pack deduplication, merging, splitting, and cleanup |
| [PySockAPI](pysockapi/) | Tool | Socket module to send commands to an MT2 server via adminpage |
| [Sequence Generator](sequence-generator/) | Tool | Auto-generate sequence tables for NetStream.cpp and sequence.cpp |
| [XTEA Keys Generator](xtea-keys-generator/) | Tool | Auto-generate random XTEA keys and EterPack.cpp array code |
| [Colored Quest Scrolls](how-to-colored-quest-scrolls/) | Tutorial | Customizable quest UI with colored text, icons, and blink effects |
| [Compile SLN from Command Line](how-to-compile-sln-from-command-line/) | Tutorial | Batch script for MSBuild compilation from CLI |
| [MakePack Type4](how-to-makepack-type4/) | Tutorial | Generate type4 packs using MakePack.exe |
| [Manually Debug Crashes with VS](how-to-manually-debug-crashes-with-vs/) | Tutorial | Step-by-step Visual Studio debugging setup |
| [Recalculate MySQL Item IDs](how-to-recalculate-mysql-item-ids/) | Tutorial | Fix "NO MORE ITEM ID RANGE" errors with SQL |
| [Translate Source Strings](how-to-translate-source-strings/) | Tutorial | Auto-extract locale strings from source code |
| [Quest Syntax Highlighting](quest-syntax-highlighting/) | Tutorial | Notepad++ syntax highlighting for MT2 quest Lua scripts |
| [WorldEditor DPI Fix](worldeditor-dpi-fix/) | Tutorial | Fix DPI scaling issues in WorldEditor |
